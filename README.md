### Creation process
```
安装
npm install -g express-generator
查看版本
express --version
创建项目（3选1）
1.express 目录名(项目名)    默认使用jade 模板引擎
2.express -e 目录名    使用ejs模板引擎
3.express --hbs 目录名 使用handlebars模板引擎

项目创建成功之后
  install dependencies:
     > cd pro1 && npm install

   run the app:
     > SET DEBUG=pro1:* & npm start

还需要安装依赖模块
cd pro1 && npm install

运行
```
